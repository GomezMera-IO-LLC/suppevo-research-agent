## How These Variants Converge to Cause Dementia

```
APOE ε4 (primary driver)
│
├─→ Impaired amyloid-beta (Aβ) clearance from brain interstitial fluid
│   ├─ ApoE4 protein binds Aβ poorly vs ApoE3 → less receptor-mediated clearance
│   └─ PICALM CC (homo) → reduced endocytic uptake of Aβ at blood-brain barrier
│
├─→ Enhanced amyloid plaque deposition
│   ├─ CLU CT → less clusterin-mediated Aβ chaperoning
│   └─ CD33 AC → microglia less efficient at phagocytosing plaques
│
├─→ Tau pathology (neurofibrillary tangles)
│   └─ BIN1 GG (homo) → dysregulated tau spreading between neurons
│       (BIN1 is the #2 AD risk gene; involved in tau propagation via endosomes)
│
├─→ Neuroinflammation (chronic, self-amplifying)
│   ├─ IL-6 CC (homo) → constitutively high IL-6 in brain
│   ├─ Activated microglia release more pro-inflammatory cytokines
│   └─ Creates feed-forward loop: inflammation → neuronal damage → more inflammation
│
├─→ Mitochondrial dysfunction
│   ├─ TOMM40 AG + AA → impaired mitochondrial protein import in neurons
│   └─ Neurons are extremely dependent on mitochondrial ATP → energy failure
│
├─→ Cerebrovascular damage (mixed dementia component)
│   ├─ MTHFR AG → ~35% reduced enzyme → moderately elevated homocysteine
│   ├─ Homocysteine is directly neurotoxic (NMDA receptor excitotoxicity)
│   └─ Homocysteine damages cerebral blood vessel endothelium
│
= Progressive cognitive decline: Amyloid + Tau + Inflammation + Vascular + Mitochondrial
```
